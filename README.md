# Pardus PM Add-On

In order to run the Pardus PM Add-on program, you need to do these steps:

 - Git clone the repository via `sudo apt install git && git clone --depth 1 https://github.com/ENux-Distro/Pardus_PM_Add-on.git`
 - Change your directory to the repositories directory via `cd /path/to/Pardus_PM_Add-on`
 - Run `bedrock-install.sh`
 - After the script reboots itself, run the Pardus PM Add-on tools via `cd /path/to/Pardus_PM_Add-on && sudo bash pardus-pm-addon`
 - Select the package managers you want to add. After you write them down, the script will do it's thing

After everything has went sucesfully, you now have installed the Pardus PM Add-on tool
